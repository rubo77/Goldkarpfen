Goldkarpfen

Goldkarpfen is a minimalistic, pure p2p communication system without any central authority,
which can utilize any transport layer that can tranport files from one node to another (support for i2p and tor is build in)
It uses a primitive, robust, text-based distributed database (ITP-FORMAT)
